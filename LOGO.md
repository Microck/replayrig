# Logo Prompts for Repatrol

**Project Context**: Multi-agent game QA swarm where AI agents (Explorer, Chaos, Reporter) autonomously play games to find bugs, capture video evidence, and file GitHub issues.

**Suggested Vibe**: Technical, autonomous, surveillance-intelligent, gaming-adjacent

---

## Color Palette

| Role | Color | Hex | Rationale |
|------|-------|-----|-----------|
| Primary | Night Radar Green | `#0DFFB3` | Evokes surveillance/night vision, autonomous patrolling agents scanning for bugs in the dark |
| Secondary | Alert Amber | `#FF6B35` | Warm discovery color — signals bug found, issue detected, the "aha" moment in testing |
| Accent | Deep Slate | `#1A1F2E` | Technical foundation, dev-tool aesthetic, the codebase beneath the swarm |
| Neutral | Ghost Cyan | `#4AFFF3` | Digital intelligence glow, the autonomous agent "thinking" color |

**Palette Story**: The palette captures the essence of autonomous agents patrolling game systems at night. Night Radar Green represents the constant scanning/surveillance. Alert Amber is the moment of discovery — when a bug is found. Deep Slate grounds it in developer-tool territory. Ghost Cyan adds the intelligence/autonomous agent glow.

## Usage Notes

- **Primary applications**: GitHub repo, demo video intro, CLI tool icon
- **Flat design preferred** — works at small sizes, reproducible
- **High contrast** — Night Radar Green on white is highly visible
- **Scalable** — geometric forms work from favicon to banner
- **Dark mode**: Invert Deep Slate as background, use Night Radar Green as foreground
